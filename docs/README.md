# Screenshots for the README

Drop PNG screenshots here with these **exact filenames** (the main README references them):

| Filename | Capture this view |
|----------|-------------------|
| `hero.png` | The **Dashboard** (dark mode recommended) — the top banner of the README. Make it wide/landscape. |
| `dashboard.png` | The **Dashboard** — stat cards, run-breakdown chart, recent executions. |
| `workflow-builder.png` | A **workflow detail** page — the steps list on the left + the Run panel / Webhook card on the right. |
| `execution-detail.png` | An **execution detail** page — the per-step logs with AI summary/classification (the money shot). |
| `connectors.png` | The **Connectors** page — active + "coming soon" cards. |

## Tips
- Use a clean browser window (hide bookmarks bar / extensions).
- 1440–1600px wide looks crisp on GitHub; keep each file under ~1 MB.
- `hero.png` and `dashboard.png` can be the same shot if you like.
- Dark mode reads especially well for a hero image.
